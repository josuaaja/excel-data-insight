# excel-data-insight

## 📌 Project Overview
This repository contains a retail sales data analysis initiative focused on extracting actionable business insights from historical data. The analysis is designed to support strategic decision-making by presenting relevant Key Performance Indicators (KPIs) aligned with operational and sales targets.

## 📂 Data Source
The raw data used in this analysis is sourced from a public Kaggle dataset:
* **Dataset Link:** [Superstore Sales Dataset](https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting?resource=download)
* **Data Context:** E-commerce and retail transactions covering customer order details, product categories, distribution regions, and profitability margins.

## 🛠️ Methodology & Tools
Data processing and analysis were entirely conducted using Microsoft Excel, leveraging the following advanced features:
* **Power Query:** Used for the Extract, Transform, Load (ETL) process. This step includes cleaning raw data, handling anomalies, and standardizing date and text formats to ensure the data is ready for analysis.
* **Pivot Tables:** Utilized for multidimensional data aggregation, summarizing sales metrics, and identifying performance trends across geographic regions and product categories.
* **Charts:** Applied for data visualization to synthesize complex numbers into easily digestible information through an interactive dashboard.

## 📂 Sumber Data
Data mentah yang digunakan dalam analisis ini bersumber dari dataset publik Kaggle:
* **Tautan Dataset:** [Superstore Sales Dataset](https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting?resource=download)

## 🛠️ Metodologi & Alat
Pengolahan dan analisis data sepenuhnya dilakukan menggunakan Microsoft Excel dengan memanfaatkan fitur-fitur tingkat lanjut berikut:
* **Power Query:** Digunakan untuk proses *Extract, Transform, Load* (ETL). Langkah ini mencakup pembersihan data mentah, penanganan nilai yang anomali, serta standardisasi format tanggal dan teks agar siap dianalisis.
* **Pivot Table:** Digunakan untuk agregasi data multidimensi, merangkum metrik penjualan, serta mengidentifikasi tren kinerja berdasarkan wilayah geografis dan kategori produk.
* **Grafik (Charts):** Visualisasi data untuk merangkum angka menjadi informasi yang mudah dipahami melalui *dashboard* interaktif.
